# Regal Revelry

A website made in ReactJS allowing event planners to look for events being held, along with the event organizer. This website may seem to target only the planners' issues, but it also makes it easier for the organizers.

The organizer has to just upload upload their event along with the bare minimum details we want them to provide, the planner can then contact them and then help them.

The website boasts a stunning UI and high efficiency by using techniques such as lazy loading.
